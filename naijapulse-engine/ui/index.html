Pipeline Health

NaijaPulse is a snapshot, not a live stream — it reflects the last pipeline run. We show our own data quality openly, because trust here should be earned by disclosure, not assumed.

Methodology

How we cluster a Story

Articles are embedded with a local model, near-duplicates collapsed to one canonical article, then canonical articles about the same real-world event are clustered into a single Story. One Story = one event, across every outlet that covered it.

The four-lean taxonomy

Pro-government, anti-government, mixed, and independent — a richer read on the Nigerian media landscape than a binary left/right split. Leans are set at the outlet level from ownership and editorial pattern, reviewed periodically, and shown as "unknown" rather than guessed when we don't have enough signal.

How to read this

Bias tags reflect ownership lean, not fact-checking. The spectrum shows how today's coverage splits across pro-government, anti-government, mixed, and independent outlets.
